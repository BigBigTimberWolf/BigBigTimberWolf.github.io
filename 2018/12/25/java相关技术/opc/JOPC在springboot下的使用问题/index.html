使用jopc在javase下使用正常、可以正常访问数据，但是放到项目上(Springboot)时，却出现了两个问题：

1.Property file javafish.clients.opc.JCustomOpc doesn’t exist. System terminat

这个是提示找不到JCustomOpc.properties 。

打开jeasyopc.jar下面的javafish.clients.opc.property.PropertyLoader.class

而我明明创建了javafish\clients\opc\JCustomOpc.properties

那么解决方案：

在pom.xml中添加：

2. java.lang.UnsatisfiedLinkError: sJCustomOpc.dll: Can’t load IA 32-bit .dll on a AMD 64-bit platform

更换JDK为32位、Jopc不支持64位

3. java.lang.UnsatisfiedLinkError: no JCustomOpc in java.library.path

没有找到你指定的dll。

一路追代码，我找到了加载dll的地方javafish.clients.opc.JCustomOpc 里面有这么一句话：
