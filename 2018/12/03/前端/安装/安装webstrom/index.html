WebStorm

​ WebStorm 是jetbrains公司旗下一款JavaScript 开发工具。目前已经被广大中国JS开发者誉为“Web前端开发神器”、“最强大的HTML5编辑器”、“最智能的JavaScript IDE”等。与IntelliJ IDEA同源，继承了IntelliJ IDEA强大的JS部分的功能。

下面将描述WebStorm 的安装过程

1. 下载WebStrom

WebStrom的官方网址:https://www.jetbrains.com/webstorm/

点击DOWNLOAD下载

2. 安装

下一步

选择路径

32位的系统就选择32-bit launcher，64位系统随便选

.js,.cc,.html,.json都勾选上

点击安装（Install）

选择I want to manually reboot late

点击Finish

到此安装完毕

3 初始化

3.1 打开WebStrom

如果你之前安装过WebStrom或者别人给你的有settings（那应该不需要看这个教程）就导入，我选择不导入设置（选择Do not import settings）

3.2 选择主题

喜欢白色主题、就选择Light，喜欢黑色就选择Darcula

点击Next:Featured plugins

3.3 选择安装的插件

一般情况下不翻墙无法下载的，直接点击Start using WebStrom

4.4 Activation（激活）

下面要进行的就是软件激活了，如果不激活可以试用30天，试用时间过了之后还要激活，所以就在这里直接激活了吧。

激活的方式有很多种，我选择了最一劳永逸的方式：补丁破解

4.4.1 下载破解补丁

破解补丁下载网站：http://idea.lanyus.com/

点击下载破解补丁

下载完毕之后，将补丁复制到webstrom安装目录的bin目录下（找不到的话可以右键点击桌面上的webstrom图标->打开文件所在位置）

修改同目录下的 WebStorm.exe.vmoptions 和WebStorm64.exe.vmoptions，这两个文件一个是32位的，一个是64位的，建议同步修改。

用文本编辑器打开之后，在文件最上面加一行代码 ：

记住两个文件都修改（避免有时候打开32位客户端有时候打开64位客户端出现没有破解的情况）

修改完毕之后再次打开webstrom

选择Activation code

4.4.2 获取注册码

打开http://idea.lanyus.com/

点击获得注册码

复制注册码

粘贴到webstrom中

5. 破解成功

点击OK，进入WebStrom，破解成功~
